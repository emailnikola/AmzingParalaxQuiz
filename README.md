# AmzingParalaxQuiz

Demo project setup with:
Vue SSR Vuetify Pinia TypeScript

## Project Setup

```sh
npm install
```

### Compile for Development

```sh
npm run dev
```

### Lint using ESLint

```sh
npm run lint
```

### Check for TypeSctipt errors

```sh
npm run type-check
```

### Run Unit Tests and Vitest and exit

```sh
npm run test
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
